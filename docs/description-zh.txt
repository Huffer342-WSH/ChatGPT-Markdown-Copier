ChatGPT Markdown Copier 用于在 ChatGPT 页面中为助手回复提供更可靠的 Markdown 复制能力。安装后，用户可以在原有回复操作区附近看到额外的“复制 Markdown”入口，一键将当前回复内容复制为更适合保存、分享、二次编辑和粘贴到笔记工具、文档系统、代码仓库或知识库中的 Markdown 文本。

这个扩展特别适合经常整理 AI 回复内容的用户，例如需要保存技术解答、整理学习笔记、归档研究资料、迁移代码示例，或把回答同步到支持 Markdown 的编辑器和协作平台的用户。相比普通复制方式，本扩展更关注复制结果的可用性与准确性，尤其针对数学公式、代码块、表格、链接、标题、列表、引用和行内代码等常见结构进行了处理，尽量减少复制后格式混乱、公式界定符错误、代码结构损坏或内容难以直接复用的问题。

项目当前的核心目标是提升用户可感知的复制质量，尤其重视数学公式和代码块的正确转换。对于包含 KaTeX 渲染公式的回复，扩展会尽量还原更适合 Markdown 使用的公式内容；对于代码块，会尽量保留换行、代码主体和常见语言信息；对于表格、链接及常见正文结构，也会按可读、可继续编辑的方向输出。这样可以帮助用户更顺畅地把 ChatGPT 回复纳入自己的工作流，而不必在复制后进行大量手工清理。

本扩展仅作用于 ChatGPT 页面中的助手回复区域，不提供外部云服务，也不会依赖额外站点来完成转换。它主要基于页面当前渲染出的内容进行处理，因此在 ChatGPT 页面结构发生调整时，个别场景的复制效果可能受到影响。我们会持续维护兼容性，并欢迎用户反馈特殊页面结构、异常复制结果或边缘案例。
